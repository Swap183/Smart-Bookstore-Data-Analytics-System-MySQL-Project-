**Smart Bookstore Data Analytics System (MySQL)**

Description:
A MySQL-based data analytics system for managing and analyzing bookstore data. The project performs data retrieval, aggregation, reporting, normalization, and dashboard-ready views.

Features:
• Data filtering by category, price, rating
• Aggregation reports (average price, ratings, reviews)
• Advanced SQL reporting queries
• Schema normalization with foreign keys
• Analytical views for dashboards

Tools Used:
MySQL, SQL

Database Tables:
• books
• publishers
• categories

Key SQL Concepts:
JOIN, GROUP BY, CASE, Subqueries, Views, Aggregation, Normalization
